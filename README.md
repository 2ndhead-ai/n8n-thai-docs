# n8n - เครื่องมือช่วยสร้างระบบอัตโนมัติสำหรับเวิร์กโฟลว์

## เกี่ยวกับ n8n
n8n (อ่านว่า "เอ็น-เอท-เอ็น") เป็นเครื่องมือที่ช่วยให้คุณสามารถเชื่อมต่อแอปพลิเคชันและบริการต่างๆ เข้าด้วยกันผ่าน API และจัดการข้อมูลได้โดยใช้โค้ดเพียงเล็กน้อยหรือไม่ต้องเขียนโค้ดเลย น8n เป็นซอฟต์แวร์โอเพนซอร์สภายใต้ใบอนุญาต [fair-code](https://faircode.io/) ที่รวมความสามารถด้าน AI เข้ากับการทำงานอัตโนมัติในกระบวนการทางธุรกิจ

## คุณสมบัติหลักของ n8n

- **ปรับแต่งได้อย่างยืดหยุ่น**: สร้างเวิร์กโฟลว์ที่ซับซ้อนได้ตามต้องการและสามารถพัฒนาโหนดแบบกำหนดเองได้
- **รองรับการเชื่อมต่อหลากหลาย**: เชื่อมต่อกับบริการและแอปพลิเคชันกว่า 400+ รายการเช่น Google Sheets, Telegram, MySQL, Slack และอื่นๆ อีกมากมาย
- **การประมวลผลข้อมูล**: จัดการและแปลงข้อมูลระหว่างบริการต่างๆ ได้อย่างมีประสิทธิภาพ
- **ความเป็นส่วนตัวและความปลอดภัย**: สามารถติดตั้งบนเซิร์ฟเวอร์ส่วนตัวเพื่อความปลอดภัยและการควบคุมข้อมูลที่ดีขึ้น
- **ทางเลือกในการใช้งาน**: ใช้ n8n ผ่าน npm หรือ Docker เพื่อทดลองใช้งาน หรือเลือกใช้บริการบนคลาวด์หากต้องการให้ n8n จัดการโครงสร้างพื้นฐานให้
- **ความสามารถด้าน AI**: รวมฟีเจอร์ AI ในเวิร์กโฟลว์เพื่อเพิ่มประสิทธิภาพการทำงานอัตโนมัติ

## วิธีการติดตั้ง

n8n มีหลายวิธีในการติดตั้งและใช้งาน:

### 1. ใช้งานผ่าน npm

```bash
npm install n8n -g
n8n start
```

### 2. ใช้งานผ่าน Docker

```bash
docker run -it --rm \
  --name n8n \
  -p 5678:5678 \
  -v ~/.n8n:/home/node/.n8n \
  n8nio/n8n
```

### 3. ใช้บริการ n8n Cloud

สำหรับผู้ที่ไม่ต้องการจัดการเรื่องโครงสร้างพื้นฐาน สามารถใช้บริการ [n8n Cloud](https://www.n8n.io/) ซึ่งจะดูแลเรื่องการติดตั้งและการดูแลระบบให้ทั้งหมด

## คู่มือการใช้งานเบื้องต้น

1. **เริ่มสร้างเวิร์กโฟลว์**: หลังจากติดตั้ง n8n แล้ว เข้าสู่หน้า dashboard ที่ `http://localhost:5678`
2. **เพิ่มโหนด**: คลิกที่ปุ่ม "Add node" เพื่อเลือกบริการที่ต้องการเชื่อมต่อ
3. **กำหนดการทำงาน**: ตั้งค่าการทำงานของแต่ละโหนด และเชื่อมโยงโหนดเข้าด้วยกันเพื่อสร้างเวิร์กโฟลว์ที่สมบูรณ์
4. **ทดสอบและเริ่มใช้งาน**: ทดสอบการทำงานของเวิร์กโฟลว์ และเมื่อพร้อมแล้วให้เปิดใช้งานเวิร์กโฟลว์

### ตัวอย่างการใช้งาน
- รับข้อมูลจาก Google Sheets และส่งการแจ้งเตือนผ่าน Telegram
- ดึงข้อมูลจาก API และบันทึกลงฐานข้อมูล MySQL
- สร้าง webhook รับข้อมูลและประมวลผลด้วย AI ก่อนส่งต่อไปยัง Slack
